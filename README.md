# host-sjb-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容独立，不依赖于特定域名或具体网站，便于直接通过 GitHub Pages 进行访问与分享。

## 目录说明

```
.
├── index.html          # 仓库首页 / 页面导航（如有）
├── pages/              # 存放独立 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── README.md           # 本说明文件
```

- 根目录下的 `index.html` 可作为页面索引或默认入口。
- `pages/` 目录集中存放各个 HTML 页面文件，可根据需要新增或更新。

## 页面归档说明

- 所有 HTML 页面均为独立文件，彼此无强依赖关系。
- 页面中引用的资源（如 CSS、JavaScript、图片等）建议内联或使用相对路径，以确保在归档环境下正常加载。
- 页面内容应保持静态，不依赖后端服务或动态数据源。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在 `pages/` 目录中添加或修改 HTML 文件。
3. 通过 GitHub Pages 功能启用发布，即可通过对应 URL 访问各页面。

## 维护说明

- 新增页面时，请将 HTML 文件放入 `pages/` 目录，并保持文件名简洁易读。
- 如需更新已有页面，直接覆盖原文件即可。
- 不建议在仓库中存放敏感信息或动态脚本，避免安全风险与兼容性问题。

## 许可

本仓库内容仅供归档与展示使用。具体页面的版权与许可请参见各文件头部注释或相关说明。
